# saec
Demonstração da Integração via WebService entre Cartórios e SAEC

[![Python Version](https://img.shields.io/badge/python-3.9-brightgreen.svg)](https://python.org)
[![Flask Version](https://img.shields.io/badge/flask-2.0-brightgreen.svg)](https://flask.palletsprojects.com/en/2.3.x/)

## Executando o Projeto Localmente

Para rodar este projeto em sua máquina local, siga os passos abaixo:

### 1. Clone o Repositório

Primeiro, clone este repositório para sua máquina local usando o comando Git:

```bash
git clone git@github.com:mupisystems/saec.git
```

### 2. Instale os componentes:

```bash
pip install -r requirements.txt
```

### 3. Rode o servidor local:
```bash
flask run
```
### 4. Acesse a documentação do WSDL:
http://127.0.0.1:5000/?wsdl
